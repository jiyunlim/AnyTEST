IMEI 확인방법

국내제조사 외산단말은 2012년 5월 1일이후 출시 단말부터 확인 가능합니다.

Android O/S

제조사별로 단말 가공에 따라 일부 차이 있을 수 있음

[New SKT 우리카드 프로모션]

S/W : 설정(Settings) → 휴대폰 정보(About phone) → 상태(Phone identity) → IMEI
H/W : 배터리 슬롯의 S/N 표기 바코드와 함께 표기(단, 일체형 스마트폰은 미표기되어 있을 수 있음)
단말박스 : S/N 등이 표기된 박스 바코드 스티커에 표기

[Wi-Fi MAC address 확인방법]

※ Wi-Fi MAC address는 S/W에서만 확인 가능합니다.
S/W : 설정(Settings) → 휴대폰 정보(About phone) → 단말정보(Hardware information) → Wi-Fi MAC address

iOS

출시단말 종류에 따라 일부 차이 있을 수 있음

S/W : 설정(Settings) → 일반(General) → 정보(Information) → IMEI / Wi-Fi MAC address
H/W : 단말기 후면 하단에 표기(H/W 표기 : 아이폰은 아이폰5부터 적용, 아이패드는 기존 모델에도 적용)
단말박스 : S/N 등이 표기된 박스 바코드 스티커에 함께 표기

Windows Mobile

출시단말에 따라 일부 차이 있을 수 있음

[IMEI 및 Wi-Fi MAC address 확인방법]

메뉴에서 Address 숫자가 보이지 않는 경우에는 Wi-Fi On 시킨 후 확인
S/W : 설정(Settings) → 정보(About) → 상세정보(More information) → IMEI
H/W : 배터리 슬롯의 S/N 표기 바코드와 함께 표기 (단, 일체형 스마트폰은 미표기되어 있을 수 있음)
단말박스 : S/N 등이 표기된 박스 바코드 스티커에 함께 표기
