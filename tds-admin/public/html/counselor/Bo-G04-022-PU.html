Check point

  • OK캐쉬백 포인트와 신용카드 포인트(KB국민카드, 우리카드)를 합산 사용 중인 고객의 경우 OK캐쉬백 포인트와 합산 사용중인 신용카드 포인트를 동시에 사용하실 경우 T모아쿠폰 정상 발급이 불가합니다.
  • 신용카드 포인트(KB국민카드, 우리카드)를 합산하여 사용 중인 고객께서는 OK캐쉬백 포인트만 사용 부탁 드립니다.
  • 조회된 신한카드 포인트는 현재 사용 가능한 신한카드를 소지하신 경우에만 사용 가능합니다. 소지하신 신한카드가 사용 불가한 상태인 경우 포인트가 조회 되더라도 정상적인 사용이 불가합니다.
  • 매일 00:00 ~ 00:30까지(30분간) 현대카드 일자전환 작업으로 인해 현대카드 M포인트 사용이 제한됩니다.
  • 제휴 포인트/마일리지 사용은 각 제휴사의 포인트정책에 따라 사용 가능합니다.
  • 포인트할인 적용 완료 후 개통취소로 인한 포인트/마일리지 사용취소는 개통취소와 함께 자동 취소처리 됩니다.
  • 포인트할인 적용 완료로 사용된 포인트/마일리지 취소처리는 제휴사에 따라 최대3~5일 소요될 수 있습니다.
